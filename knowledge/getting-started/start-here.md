# Start here

This guide gives you the shortest credential-free path through TrashPal.

Prerequisite: [understand what TrashPal demonstrates](../index.md).

## Follow the learning path

1. [Improve your first automation](improve-your-first-automation.md) and inspect the local evidence.
2. [Run the local stack](run-locally.md) to use the same behavior through the product surface.
3. Follow the Concepts sequence, then compare Homecoming with Scheduled Hauler Access.
4. Continue through HTTP and MCP, agent instrumentation, PostHog export, evaluation, and generated reference.
5. Run the repository quality gate after the focused examples pass:

   ```bash
   pnpm check
   ```

## What success looks like

You can name the component that owns approval, operation identity, and success; every command you cite has an executable owner; and no credential was needed or transmitted.
